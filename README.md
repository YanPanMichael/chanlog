# chanlog

> A beautiful javascript tool to change log and generate CHANGELOG.md file

## ✨ Feature

- fast to generate CHANGELOG.md

## 🚀 Start

### install

```bash
npm i -D chanlog # OR yarn add -D chanlog
```

```js
  "scripts": {
    "chanlog": "chanlog"
  },
```

```bash
npm run release # OR yarn release
```
